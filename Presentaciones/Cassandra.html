Conceptos básicos¶

  • Base de datos tipo clave-valor y orientada a columnas
  • Pensado para cuando el almacenamiento es barato y podemos tener duplicado los datos.
  • Se crea una tabla por cada consulta

Modelo de datos¶

Keyspace: contenemos ms exterior. Anloga a una base de datos en base de datos relacionales

Familia de columnas: colleccion de filas, y cada fila es una coleccin de columnas. Anloga a una tabla en RDBMS

Columna: Unidad bsica de datos en Cassandra.

Clave primaria¶

  • Clave particin: es la primera parte de la clave primaria. Responsable de distribuir los datos a lo largo de todos los nodos disponibles.
  • Clave compuesta: la clave primaria puede tener ms de una coulmna.
  • Clave clster: segunda parte de la clave primaria. Sirve para ordenar los datos ascendentemente descendentemente.

import os

def parse_date(date):
    date_as_string = str(date)
    year_as_string = date_as_string[6:10]
    day_as_string = date_as_string[3:5]
    month_as_string = date_as_string[0:2]
    return str(year_as_string+"-"+month_as_string+"-"+day_as_string)

def parse_time(time):
	time_as_string = str(time)

#Nmero incidencias segn el distrito'
def get_num_incidents_by_district(district):
	num_incidents = session.execute('SELECT COUNT(*) FROM incidencias.incidenciasbyzona WHERE pddistrict=%s',[district])
	return num_incidents[0].count


#listar todos los distritos
def get_districts():
	rows_districts = session.execute('SELECT pddistrict FROM distritos')
	districts = []
	for district in rows_districts:
   		districts.append(district.pddistrict)
	return districts


#Nmero incidencias segn la categoria entre dos fechas'
def get_num_incidents_by_category_betweenDate(category,start_date,end_date):
	num_incidents = session.execute('SELECT COUNT(*) FROM incidencias.incidenciasbycategoriafecha WHERE Category=%s AND date >= %s AND date <= %s',[category,start_date,end_date])
	return num_incidents[0].count


#ver todas las categorias
def get_categories():
	rows_categories = session.execute('SELECT category FROM categorias')
	categories = []
	for category in rows_categories:
   		categories.append(category.category)
	return categories
